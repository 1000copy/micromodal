Introduction

Micromodal is a modern dependency-free library for creating accessible modal dialogs. Written in ES6, it weighs only 1.6kb.

Out of the box, it allows you to create modal dialogs which conform to the WAI-ARIA guidelines and implement the interactive functionalities recommend by it.

Designed to be easy to use, it does most of the heavy lifting behind the scenes and exposes a simple api to interact with the dom.

Following are some of the interactions handled by the library:-

  • Closes modal on overlay click
  • Closes modal on esc button press
  • Toggles aria-hidden attribute
  • Traps tab focus within the modal
  • Maintains focus position while toggling modal
  • Focuses on first focusable element within modal
