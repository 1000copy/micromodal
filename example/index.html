Micromodal.js

Micromodal is a dependency-free, lightweight library for creating accessible modal dialogs

It allows you to create modal dialogs, which conform to the WAI-ARIA standards and provides the following abilities out of the box:-

  • Closes modal on overlay click
  • Closes modal on esc button press
  • Toggles aria-hidden attribute
  • Traps tab focus within the modal
  • Maintains focus position while toggling modal
  • Focuses on first focusable element within modal
